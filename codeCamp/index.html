Code Camp 2015

Brought to you by Codify Naija and Paradigm Initiative Nigeria (PIN)

Begining from July 27th-August 17th 2015, Codify Naija and Paradigm Initiative Nigeria will organize a programming camp for secondary school students. We believe in the transformative power of programming education and offer this opportunity to secondary students for free. Please refer to the application form below to apply. The camp will be rigorous, exciting and fun! Get excited!

Activities

The participants of summer code camp will be exposed to a wide-range of enriching programs. Participants will imbibe industry-applicable skills which include:

  • Web design in html and css
  • Software development in Javascript
  • Debugging code in software development
  • Team-building and leadership skills
